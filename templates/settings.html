使用说明

快速入门

1. 在「分类管理」中新建分类并添加视频所在目录
2. 服务器会自动扫描目录中的视频文件并生成缩略图
3. 首页即可看到所有视频，点击即可播放

视频识别规则

  • 合集 — 目录下无直连视频，但有子目录含视频，子目录递归扫描生成独立卡片
  • 扁平 — 目录下有视频文件且无含视频的子目录，每个视频独立显示
  • 混合 — 目录下既有直连视频又有含视频的子目录，直立视频独立卡片 + 子目录递归
  • 目录标签上的彩色徽章标识目录类型，方便区分
  • 支持的格式：MP4（直接播放）、MKV/AVI/RMVB/FLV 等（需开启自动转码后转为 MP4）

设置页卡片

设置页共 5 个可折叠卡片：分类管理、高级设置、后台任务、数据管理、使用说明。点击卡片标题可展开/收起，有活跃任务时会自动展开对应卡片。所有开关切换后自动实时保存，数值类设置需点击「保存设置」按钮生效。

自动扫描

位于「高级设置」卡片顶部。开启后，服务器会自动监控目录变化、生成缩略图和执行转码任务。关闭可节省系统资源，适合低配设备或长时间不添加新视频时使用。

转码设置

位于「高级设置」卡片中，包含三个开关：

  • 自动转码为 MP4 — 开启后，扫描到非 MP4 视频时自动加入转码队列；关闭时仅标记不转码（默认开启）
  • 转码后删除原文件 — 开启后节省磁盘空间，关闭则保留原始文件（默认开启）
  • GPU 硬件转码（NVENC） — 使用 NVIDIA 显卡加速转码，速度快 5~10 倍（需 NVIDIA 显卡 + 支持 NVENC 的 FFmpeg，未检测到时自动隐藏）

转码支持两种模式：CPU 软编（libx264，兼容性好）和 GPU 硬件转码（NVIDIA NVENC）。仅浏览器兼容的 MP4（H.264/H.265/VP9/AV1 编码）可直接播放。以下情况会触发转码：

  • 非 MP4 格式（MKV、AVI、RMVB、FLV 等）→ 自动加入后台转码队列
  • 非标 MP4 文件（如 MPEG-4 Part 2 / mpeg4 编码）→ 浏览器无法解码，自动标记为需转码

后台任务

合并了转码管理和元数据管理，展开后可查看转码队列状态（任务数、等待中、转码中）、取消转码任务，以及元数据补全进度和手动补全操作。

数据管理

合并了缓存清理和隐藏视频管理。可按需清理缩略图/元数据/系列缓存，或一键清理全部缓存。隐藏视频管理支持恢复或彻底删除。

分类管理

  • 重新扫描 — 检测目录变化并增量更新，只更新有改动的内容，保留原有数据。也可在首页点击分类图标快速扫描。自动扫描开启时无法使用
  • 导出 / 导入 — 备份与恢复所有配置和缓存（config.pb、分类、缩略图缓存、视频元数据缓存、隐藏系列、系列缓存），迁移到新电脑时可用，无需重新扫描。自动扫描开启时无法使用
  • 清空 — 清除所有分类配置，清空后可导入备份恢复
  • 新建分类 — 自定义名称、颜色和图标，分类下可添加多个目录
  • 添加目录 — 浏览本地磁盘选择文件夹，也可在「未分配目录」中快速添加
  • 拖拽排序 — 拖动分类卡片可调整首页分类的显示顺序
  • 红色警告标记表示目录不存在，请及时移除

高级设置

  • 自动扫描 — 控制目录监控、缩略图生成和转码的总开关
  • 自动转码为 MP4 — 控制扫描到非 MP4 时是否自动加入转码队列（默认开启，切换后实时保存）
  • 转码后删除原文件 — 开启后节省磁盘空间，关闭则保留原始文件（切换后实时保存）
  • GPU 硬件转码(NVENC) — 使用 NVIDIA 显卡硬件加速转码，速度提升 5~10 倍（切换后实时保存）
  • 空闲检测间隔 — 后台检查缩略图缺失和转码任务的时间间隔（默认 300 秒）
  • 文件监控间隔 — 检测目录文件变化的轮询频率（默认 5 秒）
  • 缓存有效期 — 视频列表缓存的过期时间（默认 300 秒）
  • 元数据采集线程 — 并行采集线程数，越大越快但 CPU 占用越高（默认 4）
  • 详情页查询线程 — 详情页并行查询的线程数（默认 8）
  • 元数据缓存上限 — 最多缓存的视频元数据条数，超出按 LRU 淘汰（默认 5000）
  • 服务器端口 / 绑定地址 — 修改后需重启服务器生效

播放页快捷键

  • 空格 — 播放 / 暂停
  • ← / → — 快退 / 快进 10 秒
  • ↑ / ↓ — 调节音量
  • [ / ] — 上一集 / 下一集
  • F — 全屏切换

服务器状态栏

页面顶部显示服务器运行状态：版本号、运行时长、FFmpeg 状态（异常时请检查 ffmpeg 文件夹）、NVENC 状态（GPU 硬件转码是否可用/已启用）、CPU / GPU / 内存 / 磁盘占用率。鼠标悬停可查看详细信息。数据每 5~10 秒自动刷新。
